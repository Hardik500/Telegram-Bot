# Telegram-Bot
A telegram bot for fetching Cyanide &amp; Happiness comics from explosm.net

## Install

1. Clone the repository with :
```
git clone https://github.com/Hardik500/Telegram-Bot.git
```
2. Edit the token at line 11 with your token
3. Run the bot with
```
python bot.py
```
